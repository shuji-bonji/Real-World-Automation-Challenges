## 3. 自動化全般に関する社会的・倫理的・実装上の課題
- 法的枠組みの未整備
- 倫理的ガイドラインの欠如
- 実装における課題とリスク

→ [詳細はこちら](./social-ethical-and-implementation-challenges-of-real-world-automation.md)

## AGIの影響について

AGI（汎用人工知能）の進展により、これらの課題の一部は新たな形で再定義される可能性があります。  
技術の進化が社会や倫理に与える影響についても継続的に検討していく必要があります。
